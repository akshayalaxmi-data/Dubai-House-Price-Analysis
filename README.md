# Dubai House Price Analysis Dashboard

## Project Overview
This project analyzes Dubai housing market data using Power BI to identify pricing patterns, property characteristics, and market segments.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Features
- KPI Cards
- Price per Sqft Analysis
- Property Age Analysis
- Listing Category Segmentation
- Interactive Filters and Visualizations

## Key Insights
